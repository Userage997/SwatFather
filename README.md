# Сайт-лендинг "Святой Отец"

Стильный и анимированный сайт-переходник для популярного человека в Telegram-сообществе.

## Особенности

-  Темная тема с анимированным градиентным фоном
-  Плавные анимации и переходы
-  Полная адаптивность под все устройства
-  Интерактивные элементы с эффектами при наведении
-  Несколько разделов (главный, проекты, ссылки)

## Использованные технологии

- HTML, CSS3, JavaScript (без фреймворков)
- CSS Grid и Flexbox для верстки
- CSS анимации и ключевые кадры
- Google Fonts (Playfair Display, Roboto)
- Font Awesome иконки
